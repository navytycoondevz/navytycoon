## 🔧 Fix-It Guide — Troubleshooting Common Issues

### 🚫 Script won't execute / nothing happens after inject

**Cause:** Roblox process not fully loaded, or injection timing too early
**Fix:**
```bash
# Wait for game to show lobby screen fully, then re-attach
# If using macOS: restart the launcher with elevated privileges
sudo ./launcher.sh --force-attach
```
Also try: closing all browser tabs and background apps to free up RAM before injecting.

---

### 🖥️ GUI doesn't appear on screen

**Cause:** GUI rendered off-screen due to resolution mismatch
**Fix:** In config, reset GUI position manually:
```bash
GUI = { Position = {X = 20, Y = 80} }
```
Then re-execute. If still invisible, toggle fullscreen mode off and play in windowed 1280×720.

---

### 📱 Android injection fails with "process not found" error

**Cause:** Roblox launched outside the injector's managed environment
**Fix:** Always launch Roblox **from inside** the injector app, not from your home screen. Uninstall and reinstall Roblox if the issue persists after relaunching.

---

### 🍎 iOS app revoked after a few days

**Cause:** Free Apple developer certificates expire every 7 days
**Fix:** Re-sign the IPA on your PC weekly. Consider a paid Apple Developer account ($99/year) for 1-year certificates.

---

### 🔄 Auto rebirth triggering too early / too late

**Cause:** `RebirthThreshold` config value incorrect for current tycoon version
**Fix:**
```bash
AutoRebirth = true,
RebirthThreshold = 2500  -- Increase this value if triggering too early
```
Adjust until rebirth fires at your preferred progression point.

---

### 💥 Kill All crashes the game client

**Cause:** Too many simultaneous kill events overwhelming the client
**Fix:** Set `AttackRange` to `100` instead of `200`, and use `"nearest"` targeting mode rather than `"all"`.

---

### ⏳ Auto cash collection feels slow / missing nodes

**Cause:** `CollectInterval` set too high or `CollectRadius` too small
**Fix:**
```bash
CollectInterval = 0.3,   -- Lower = faster sweeps
CollectRadius = 80       -- Increase to catch distant nodes
```
